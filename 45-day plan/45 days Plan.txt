### ‚úÖ Array Operations

| Operation          | Method                | Time Complexity |
| ------------------ | --------------------- | --------------- |
| Access             | arr[i]                | O(1) ‚úÖ          |
| Insert at end      | push()                | O(1) ‚úÖ          |

| Copy               | slice()               | O(n)            |

### ‚úÖ Example Code

```js
let arr = [5, 10, 15];
arr.push(20);      // [5,10,15,20]
arr.pop();         // [5,10,15]
arr.unshift(1);    // [1,5,10,15]
arr.shift();       // [5,10,15]
```

---

### ‚úÖ Traversing an Array

```js
const arr = [10, 20, 30];

// Standard loop
for (let i = 0; i < arr.length; i++) console.log(arr[i]);

// for...of
for (let num of arr) console.log(num);

// forEach
arr.forEach(num => console.log(num));
```
