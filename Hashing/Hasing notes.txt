### **Hashing in Data Structure using JavaScript**
#### **What is Hashing?**
Hashing is a technique used in data structures to map data to a fixed-size value (hash code) using a hash function. It is primarily used for quick data retrieval. Hashing allows efficient searching, insertion, and deletion operations.

A **hash function** takes an input (key) and converts it into a numeric hash code, which is then used to store the value in a **hash table**.

#### **Why Use Hashing?**
- **Fast Lookup:** Hashing provides constant time complexity O(1) for search, insert, and delete operations in ideal conditions.
- **Efficient Data Storage:** Used for quick data retrieval in large datasets.
- **Avoids Collision:** Good hash functions minimize the chances of two different keys getting the same hash code.
- **Security:** Hashing is used in cryptography to store passwords securely.

#### **Where to Use Hashing?**
- **Databases** (Indexing, caching, lookup tables)
- **Cryptography** (Password hashing, digital signatures)
- **File systems** (Detect duplicate files using hash values)
- **Networking** (Data integrity verification using hash functions)
- **Compilers and Interpreters** (Symbol tables for variables and function names)
- **Load Balancing** (Distributing workload using hash-based partitioning)

#### **How is Hashing Beneficial?**
- Reduces the time complexity of searching operations.
- Ensures data security using cryptographic hash functions.
- Optimizes memory usage in large-scale applications.
- Prevents duplicate records in databases.

### **Hashing Example in JavaScript (Implementing a Simple Hash Table)**
```javascript
class HashTable {
  constructor(size = 10) {
    this.table = new Array(size);
  }

  _hash(key) {
    let hash = 0;
    for (let i = 0; i < key.length; i++) {
      hash += key.charCodeAt(i);
    }
    return hash % this.table.length;
  }

  set(key, value) {
    const index = this._hash(key);
    if (!this.table[index]) {
      this.table[index] = [];
    }
    this.table[index].push([key, value]);
  }

  get(key) {
    const index = this._hash(key);
    if (this.table[index]) {
      for (let pair of this.table[index]) {
        if (pair[0] === key) {
          return pair[1];
        }
      }
    }
    return undefined;
  }

  remove(key) {
    const index = this._hash(key);
    if (this.table[index]) {
      this.table[index] = this.table[index].filter(pair => pair[0] !== key);
    }
  }
}

// Example Usage
const ht = new HashTable();
ht.set("name", "Alice");
ht.set("age", 25);
console.log(ht.get("name")); // Output: Alice
console.log(ht.get("age"));  // Output: 25
ht.remove("age");
console.log(ht.get("age"));  // Output: undefined
```
